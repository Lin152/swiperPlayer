<h3>react 实现的一个简单的大图轮播组件</h3>

<h4>npm i swipeplayer  导入</h4>

<h3>使用</h3>

<h4>import Swiper from 'swipeplayer'</h4>

<h4><Swiper images={} height={} width={}/></h4>

<h3>设置参数</h3>

<h4>里面只有3个参数</h4>
<h4>images是由多个图片的路径地址所组成的数组</h4>
<h4>height是组件的高</h4>
<h4>width是组件的宽</h4>
<h4>宽高可以不设置，默认为1000px X 600px</h4>
